Maze Run – Game

Game Legend:

🔵 Blue - Player
🔴 Red - Enemy 1
🟤 Dark Red - Enemy 2 (Hard Mode)
🟢 Green - Exit
🟡 Gold - Coins
🟡 Yellow - Extra Turn
🔵 Cyan - Randomizer
⚫ Gray - Wall

Goal: Collect coins and reach the green exit while avoiding the red enemies!

Hard Mode: Face 2 enemies and more walls for the ultimate challenge!
